# Using Vite 
- faster technique
    1. npm create vite@latest
    2. enter project name:  01ViteReact
    3. enter Package name:  01vitereact
    4. select framework: React
    5. select a variant : javascript

- now run 
    1. cd 01ViteReact
    2. npm install
    3. npm run dev
